13. 把所有commit过的记录统一成一条
- 复制最后版本的项目到另一个路径，删除.git文件夹
- git reset --hard 首条记录md5   本地仓库回滚到首条记录
- git push -f orgin HEAD:master  强制提交到首条记录，即远程仓库回滚
- git branch res 
- git checkout res
- 删除除.git下所有文件，复制之前的最后版本项目到该路径下
- res 合并到 master (pull request也行)
